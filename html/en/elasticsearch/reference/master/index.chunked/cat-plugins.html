cat pluginsedit

The plugins command provides a view per node of running plugins. This information spans nodes.

GET /_cat/plugins?v&s=component&h=name,component,version,description

Might look like:

name    component               version   description
U7321H6 analysis-icu            6.0.0-alpha1 The ICU Analysis plugin integrates Lucene ICU module into elasticsearch, adding ICU relates analysis components.
U7321H6 analysis-kuromoji       6.0.0-alpha1 The Japanese (kuromoji) Analysis plugin integrates Lucene kuromoji analysis module into elasticsearch.
U7321H6 analysis-phonetic       6.0.0-alpha1 The Phonetic Analysis plugin integrates phonetic token filter analysis with elasticsearch.
U7321H6 analysis-smartcn        6.0.0-alpha1 Smart Chinese Analysis plugin integrates Lucene Smart Chinese analysis module into elasticsearch.
U7321H6 analysis-stempel        6.0.0-alpha1 The Stempel (Polish) Analysis plugin integrates Lucene stempel (polish) analysis module into elasticsearch.
U7321H6 analysis-ukrainian        6.0.0-alpha1 The Ukrainian Analysis plugin integrates the Lucene UkrainianMorfologikAnalyzer into elasticsearch.
U7321H6 discovery-azure-classic 6.0.0-alpha1 The Azure Classic Discovery plugin allows to use Azure Classic API for the unicast discovery mechanism
U7321H6 discovery-ec2           6.0.0-alpha1 The EC2 discovery plugin allows to use AWS API for the unicast discovery mechanism.
U7321H6 discovery-file          6.0.0-alpha1 Discovery file plugin enables unicast discovery from hosts stored in a file.
U7321H6 discovery-gce           6.0.0-alpha1 The Google Compute Engine (GCE) Discovery plugin allows to use GCE API for the unicast discovery mechanism.
U7321H6 ingest-attachment       6.0.0-alpha1 Ingest processor that uses Apache Tika to extract contents
U7321H6 ingest-geoip            6.0.0-alpha1 Ingest processor that uses looksup geo data based on ip adresses using the Maxmind geo database
U7321H6 ingest-user-agent       6.0.0-alpha1 Ingest processor that extracts information from a user agent
U7321H6 jvm-example             6.0.0-alpha1 Demonstrates all the pluggable Java entry points in Elasticsearch
U7321H6 mapper-murmur3          6.0.0-alpha1 The Mapper Murmur3 plugin allows to compute hashes of a field's values at index-time and to store them in the index.
U7321H6 mapper-size             6.0.0-alpha1 The Mapper Size plugin allows document to record their uncompressed size at index time.
U7321H6 store-smb               6.0.0-alpha1 The Store SMB plugin adds support for SMB stores.

We can tell quickly how many plugins per node we have and which versions.
