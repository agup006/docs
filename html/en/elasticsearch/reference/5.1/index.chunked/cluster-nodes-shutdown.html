Nodes shutdownedit

The _shutdown API has been removed. Instead, setup Elasticsearch to run as a service (see Install Elasticsearch with RPM, Install Elasticsearch with Debian Package, or Install Elasticsearch on Windows) or use the -p command line option to write the PID to a file.
